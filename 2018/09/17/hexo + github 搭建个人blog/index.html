hexo + github 搭建个人blog

Posted on 2018-09-17 |

搭建平台

  1. 安装Node.js，官网连接
  2. 安装Git
  3. 安装hexo，打开cmd命令如下:

npm install -g hexo –registry=https://registry.npm.taobao.org

$ hexo init
$ cd
$ npm install

.
├── _config.yml
├── package.json
├── scaffolds
├── source
| ├── _drafts
| └── _posts
└── themes

deploy:
type: git
repository: https://github.com/yourname/yourname.github.io.git
branch: master

$ npm install hexo-deployer-git –save
$ hexo g（本地生成静态文件）
$ hexo d（将本地静态文件推送至Github）

title 文章的标题
date 创建日期 （文件的创建日期 ）
tags 标签
categories 分类
编写正文内容（MakeDown）

$ hexo clean
$ hexo d -g
`

github做Markdown图床

在github上新建一个repository，将图片传入。

网页浏览复制图片URL，如：
https://github.com/zhanghlHUST/figureForMarkdown/blob/master/17_urllib_parse_urlparse.JPG
那么，外接连接就是将/blob/改成/raw/：
https://github.com/zhanghlHUST/figureForMarkdown/raw/master/17_urllib_parse_urlparse.JPG
