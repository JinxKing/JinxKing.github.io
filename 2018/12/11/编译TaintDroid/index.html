把源码导入到IDEA

如果make版本是3.82或者3.81，跳过这个步骤。下载3.82版本的make文件，地址是：ftp://ftp.gnu.org/gnu/make/，然后解压文件，在make的根目录下，输入命令：

$ ./configure
$ make
$ sudo make install

执行命令：

$ source build/envsetup.sh
$ mmm development/tools/idegen/

输出的结果有一行：Install: out/host/linux-x86/framework/idegen.jar。在源码目录下的out/host/linux-x86/frameworks/目录下生成了idegen.jar文件。

在源码目录下继续执行如下命令：

$ development/tools/idegen/idegen.sh

development/tools/idegen/idegen.sh命令执行完成，会在根目录下生成android.iml和android.ipr这两个文件，这两个文件是Android Studio的工程配置文件。
